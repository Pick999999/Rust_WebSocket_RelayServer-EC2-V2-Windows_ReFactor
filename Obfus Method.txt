1. 🚫 ห้ามดู Console.log (Anti-Debugging)
สามารถทำได้หลายวิธี:

Override console object - เขียนทับ console.log, console.warn, console.error ให้เป็นฟังก์ชันว่าง
Detect DevTools - ตรวจจับการเปิด DevTools แล้วหยุดทำงานหรือ redirect ออก
Debugger trap - ใส่ debugger statement ที่ทำให้คนที่พยายาม debug ติดลูปไม่รู้จบ
Timing detection - ตรวจจับเวลาที่ช้าผิดปกติ (เกิดจาก breakpoint)
2. 🌐 ห้าม Run ที่ URL อื่น (Domain Lock)
สามารถทำได้โดย:

Check window.location.hostname - ตรวจสอบว่า domain ตรงกับที่อนุญาตหรือไม่
Check document.referrer - ตรวจสอบว่ามาจากไหน
Encrypt/Decrypt based on domain - ใช้ domain เป็นส่วนหนึ่งของ key ในการ decrypt code หลัก ถ้า domain ไม่ตรง code จะไม่ทำงาน
3. 🔒 Obfuscation Techniques
จะรวมเทคนิคเหล่านี้ด้วย:

Variable/Function name mangling - เปลี่ยนชื่อตัวแปรเป็น _0x1a2b3c
String encoding - encode string เป็น hex/base64
Control flow flattening - ทำให้ logic อ่านยาก
Dead code injection - ใส่ code ปลอมๆ ที่ไม่ทำงาน
Self-defending code - code ที่ตรวจจับการ format/beautify แล้วพัง
