# 🛡️ คู่มือความปลอดภัย - Deriv Accumulator Bot

## 📌 อ่านก่อนเทรด! (สำคัญมาก)

Bot นี้ได้รับการปรับปรุงให้**ปลอดภัยที่สุด** ตามหลักการ:
> **"ขายหมูไม่เป็นไร ให้ปลอดภัยไว้ก่อน"**

---

## ⚡ สิ่งที่เพิ่มเข้ามาใหม่

### 🛡️ ระบบป้องกัน 5 ชั้น

1. **หยุดขาดทุนรายวัน** - เทรดไม่เกินที่กำหนดต่อวัน (เช่น $50)
2. **หยุดขาดทุนต่อรอบ** - แต่ละ Lot ขาดทุนไม่เกิน (เช่น $20)
3. **หยุดเมื่อแพ้ติด** - แพ้ติดกัน 5 ครั้ง = หยุดทันที
4. **ลด Stake อัตโนมัติ** - แพ้แล้วลดเงินเดิมพัน 30%
5. **ขายเร็วขึ้นเมื่อแพ้ติด** - ยิ่งแพ้มาก ยิ่งขายไว

---

## 🎮 วิธีใช้งาน (3 ขั้นตอน)

### 1️⃣ เลือกโหมดตามประสบการณ์

| โหมด | ทุนขั้นต่ำ | Stake | เสี่ยงต่อวัน | เหมาะกับ |
|------|-----------|-------|-------------|---------|
| 🧪 **ทดสอบ** | $100 | $0.50 | $5 | ทดลองใช้ |
| 🟢 **มือใหม่** | $500 | $1.50 | $25 | เริ่มต้น 0-3 เดือน |
| 🟡 **ปานกลาง** | $1,000 | $5 | $50 | มีประสบการณ์ 3-6 เดือน |
| 🔴 **ขั้นสูง** | $5,000+ | $10 | $100 | เซียนเทรด 6+ เดือน |

### 2️⃣ ตั้งค่าความปลอดภัย

**สำหรับมือใหม่ (แนะนำ):**
```
Max Daily Loss:        $25  ← ขาดทุนสูงสุดต่อวัน
Max Session Loss:      $10  ← ขาดทุนสูงสุดต่อรอบ
Max Consecutive Loss:  3    ← แพ้ติดกัน 3 ครั้ง = หยุด
Reduce Stake on Loss:  50%  ← แพ้แล้วลดเงินเดิมพัน 50%
Min Stake:             $0.50 ← Stake ต่ำสุดหลังลดแล้ว

Panic Level:           50%  ← ขายเร็วเมื่อใกล้ Barrier
Velocity Level:        40%  ← ขายเร็วเมื่อกราฟพุ่งเร็ว
```

### 3️⃣ กด Start และเฝ้าดู

- ✅ **ทำ:** เฝ้าดูหน้าจอทุก 30 นาที
- ✅ **ทำ:** บันทึก P&L ทุกวัน
- ❌ **ห้าม:** เพิ่ม Stake หลังแพ้
- ❌ **ห้าม:** เปลี่ยน Settings กลางทาง

## 💡 ตัวอย่างการทำงาน

### สถานการณ์ที่ 1: แพ้ติดกัน

**ไม่มีระบบป้องกัน (เดิม):**
```
Trade 1: $10 → แพ้ -$10
Trade 2: $10 → แพ้ -$10
Trade 3: $10 → แพ้ -$10
Trade 4: $10 → แพ้ -$10
Trade 5: $10 → แพ้ -$10
รวม: -$50 😱
```

**มีระบบป้องกัน (ใหม่):**
```
Trade 1: $10.00 → แพ้ -$10.00
Trade 2: $7.00  → แพ้ -$7.00  ← ลด 30%
Trade 3: $4.90  → แพ้ -$4.90  ← ลด 30%
Trade 4: $3.43  → แพ้ -$3.43  ← ลด 30%
Trade 5: 🛑 หยุด! (แพ้ติด 5 ครั้ง)
รวม: -$25.33 ✅ (ประหยัด 49%!)
```

### สถานการณ์ที่ 2: ขาดทุนเกิน Limit

```
09:00 - เริ่มเทรด: $0
10:30 - ขาดทุน: -$15 (ยังโอเค)
12:00 - ขาดทุน: -$30 (ยังโอเค)
14:00 - ขาดทุน: -$50 → 🛑 Bot หยุดอัตโนมัติ

📢 "Daily Loss Limit Reached! มาเทรดใหม่พรุ่งนี้นะครับ 🙏"
```

## 🎯 กฎทอง 3 ข้อ

1. **ไม่เสี่ยงเกิน 5% ต่อวัน**
   - ทุน $1,000 → เสี่ยงได้สูงสุด $50/วัน

2. **แพ้แล้วต้องลด Stake**
   - ห้ามเพิ่มเงินเดิมพันหลังแพ้! (Martingale = อันตราย)

3. **ถึงเป้าหมาย = หยุดเทรด**
   - ได้กำไรตามเป้าแล้ว → ปิดเครื่อง พักผ่อน

---

## ⚠️ สัญญาณอันตราย

หยุดเทรดทันที ถ้าเจอ:
- ❌ แพ้ติดกัน 7 ครั้งขึ้นไป
- ❌ Panic Sell บ่อยเกิน 30% ของการเทรด
- ❌ Win Rate ต่ำกว่า 45%
- ❌ ถูก Daily Limit ทุกวัน
- ❌ รู้สึกเครียด/โกรธ/กลัว
